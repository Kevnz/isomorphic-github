isomorphic github
=================

An example to test isomorphic application with react + flux + react-router + dispatcher + express

**Tools I use:**

* React - View layer
* Flux - data flow
* Dispatcher - isomorphic dispathcer
* react-router - router
* express - server
* express-state - share state to client in the back-end
* superagent - xhr library

**Build tools for dev:**

* react-hot-loader
* webpack
